# Image Editor

---

## Features

- [x] Upload Image
- [x] Save Image
- [x] Preview Image
  - [x] Before uploading
  - [x] Before downloading/saving to desktop
- [ ] Change Background Color
- [x] Download Image
  - [x] as PNG
  - [x] as JPEG
  - [x] as JPG
- [ ] Crop Image
- [ ] Resize
- [ ] Zoom Image
- [x] Colorized Effects:
  - [x] Brightness
  - [x] Contrast
  - [x] Saturation
  - [x] Clarity
  - [x] Sepia
  - [x] Black & White
  - [x] Greyscale
